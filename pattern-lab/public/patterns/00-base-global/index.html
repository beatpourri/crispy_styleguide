BEM infos

▼

*Block, E*lement, M*odifier

All CSS class names follow the BEM pattern. For Usage see .md

http://getbem.com/naming

BEM - BLOCK infos

▼

<div class="baseclass">Baseclass only</div>

<div class="baseclass baseclass--modifier1 baseclass--modifier2">Baseclass modifiers</div>

Also See: https://en.bem.info/methodology/quick-start/#block

BEM - ELEMENT infos

▼
<div class="baseclass__elementname"></div> <div class="elementname"> </div>

Also see: https://en.bem.info/methodology/quick-start/#element

BEM - MODIFIER infos

▼

<div class="baseclass__elementname baseclass__elementname--modifier01 baseclass__elementname--modifier02 baseclass"> </div> <div class="baseclass baseclass--modifier01"> <div class="class="baseclass__elementname""> </div> </div>

or/and

<div class="baseclass"> <div class="baseclass__elementname baseclass__elementname--modifier"> </div> </div>

Also see: https://en.bem.info/methodology/quick-start/#element
